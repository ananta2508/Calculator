# 🧮 Simple Calculator

## 📌 Overview

The **Simple Calculator** is a command-line application developed using **Python** that performs basic arithmetic operations through a menu-driven interface. This project was created as part of my **Python Programming Internship at CodSoft**.

The calculator allows users to perform multiple mathematical operations by entering two numbers and selecting the desired operation from the menu. It is designed with a simple and user-friendly interface, making it suitable for beginners who are learning Python programming.

This project demonstrates how Python can be used to build practical applications while applying fundamental programming concepts such as functions, loops, conditional statements, user input, and arithmetic operators.

## ⚙️ How It Works

1. The program displays a menu of available operations.
2. The user selects an operation.
3. The user enters two numbers.
4. The calculator performs the selected operation.
5. The result is displayed instantly.
6. The user can continue performing calculations until choosing to exit.
